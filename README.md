# It's a game of rock paper scissors

You can find the rules [here](https://en.wikipedia.org/wiki/Rock_paper_scissors)

# Install

```
nvm use
npm ci
```

# Build the app

```
npm run build
```

# Start the app

```
npm run start
```
or 
```
npm run start:build
```
to build then start
or 
```
npm run start:dev
```
to start the app in dev mode
# Tests

```
npm run test
```
